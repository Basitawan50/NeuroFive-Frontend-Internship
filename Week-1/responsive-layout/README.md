# Responsive Layouts - Flexbox & Grid Playground

## Project Overview

This project demonstrates the difference between Flexbox and CSS Grid by creating the same pricing section using both layout techniques.

## Features

- Semantic HTML5
- Flexbox Layout
- CSS Grid Layout
- Responsive Design
- Media Queries
- Header & Footer
- Pricing Cards
- Modern UI

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts
